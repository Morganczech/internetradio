Touch Radio is a privacy-friendly internet radio player focused on
stability, clean design, and reliable background playback.

Features:
- Internet radio playback using Media3
- Favorites and station categories
- Drag-and-drop station ordering
- Import and export stations (JSON)
- 5-band equalizer
- Sleep timer and home screen widgets
- Multi-language support (CZ, SK, EN, DE)

Privacy:
- No ads
- No tracking
- No analytics
- No location permission required
- Region is resolved using system language only

The app works offline except when streaming audio
and is fully open-source.
